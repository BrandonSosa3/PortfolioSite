Bike Share Demand Forecasting - Technical Deep Dive

Challenge: Urban bike share systems need accurate demand forecasting to optimize bike distribution and operational efficiency across city networks.

Solution: Developed a Random Forest regression model using 2 years of historical data (17,379 hourly records) from Washington DC's Capital Bikeshare system, achieving 67% prediction accuracy (R² = 0.667) with RMSE of 127 bikes per hour.

Key Results:

• Hour of day accounts for 48.6% of predictive power
• Weekday commuters are 5x more weather-resistant than weekend riders
• Rush hour periods show 3x baseline usage
• Clear weather averages 206 bikes/hour vs 62 during heavy rain
• 48-bike improvement over linear regression baseline

Technical Implementation:

• Data pipeline with Kaggle API integration
• Feature engineering for weather comfort indices and temporal indicators
• SQLite database with SQLAlchemy ORM
• Interactive Streamlit dashboard with Plotly visualizations
• Time-aware train/test split validation

CoLiv - Technical Deep Dive

Challenge: Traditional property management systems track properties at the unit level, but co-living spaces require room-level granularity with individual pricing, tenant profiles, and roommate compatibility matching.

Solution: Built a dual-portal system with separate interfaces for operators and tenants, featuring room-level tracking, individual pricing management, and an AI-powered roommate matching algorithm currently in development.

System Architecture:

• Operator Portal: Full property management from properties → units → rooms → tenants
• Tenant Portal: Personal dashboard with room details, payments, and roommate information
• Room-Level Tracking: Each room has individual pricing, size specifications, and tenant profiles
• AI Roommate Matcher: Algorithm in development for lifestyle-based compatibility matching

User Workflow:

Operator Side:
1. Create operator account at co-liv.vercel.app
2. Add property with address and details
3. Create units within the property
4. Define rooms with pricing and specifications
5. Add tenants with credentials
6. Monitor occupancy and manage operations

Tenant Side:
1. Receive credentials from operator
2. Login at co-liv-tenant.vercel.app
3. View room details and amenities
4. Access payment information
5. Connect with roommates
6. Submit maintenance requests

Key Differentiator: Unlike competitors that stop at unit-level management, CoLiv provides granular room-level control, giving operators complete visibility and flexibility across their co-living properties.
